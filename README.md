# Braille Translation

โปรแกรมแปลงข้อความภาษาไทยและภาษาอังกฤษเป็นอักษรเบรลล์ (Braille) ในรูปแบบ ASCII Art (พิมพ์ออกทางหน้าจอ 3 แถว) โดยอ้างอิงมาตรฐานตารางอักษรเบรลล์จาก [Liblouis](https://github.com/liblouis/liblouis)

## ไฟล์ในโปรเจกต์
- `BrailleTranslation_TH.py`: ตัวแปลงภาษาไทย (รองรับพยัญชนะ, สระ, วรรณยุกต์, ตัวเลขไทย, ตัวลขอารบิก และเครื่องหมายวรรคตอน)
- `BrailleTranslation_EN.py`: ตัวแปลงภาษาอังกฤษ (Grade 1 / Uncontracted รองรับตัวอักษร A-Z, ตัวเลข และเครื่องหมายวรรคตอน)

## วิธีการใช้งาน (How to run)

ต้องติดตั้ง **Python 3** ในเครื่องเพื่อรันสคริปต์ เปิด Terminal หรือ Command Prompt ไปที่โฟลเดอร์ของโปรเจกต์แล้วรันคำสั่งดังต่อไปนี้:

### 1. แปลงข้อความภาษาไทย
```bash
python BrailleTranslation_TH.py
```
เมื่อรันแล้ว โปรแกรมจะรอรับข้อความ ให้พิมพ์ข้อความแล้วกด Enter เช่น `สวัสดี`

### 2. แปลงข้อความภาษาอังกฤษ
```bash
python BrailleTranslation_EN.py
```
พิมพ์ข้อความภาษาอังกฤษแล้วกด Enter เช่น `hello`

## รูปแบบผลลัพธ์ (Output)

โปรแกรมจะแสดงอักษรเบรลล์ 1 เซลล์ ด้วยตัวอักษร 3 แถว (แถวละ 2 กล่อง) ในรูปแบบ:
- `*` = จุดนูน (dot)
- `_` = จุดสัมผัสเรียบ (no dot)

ตัวโปรแกรมจะปริ้นต์ผลลัพธ์ออกมา **2 รูปแบบ** เสมอ:
1. **ผลลัพธ์ปกติ (อ่านจากซ้ายไปขวา):** ตรงตามรูปแบบของอักษรเบรลล์สำหรับการใช้นิ้วอ่าน
2. **ผลลัพธ์แบบ Mirror (สลับซ้าย-ขวา):** ใช้สำหรับการเจาะกระดาษด้วยสเลท (Slate & Stylus) ซึ่งต้องเจาะจากด้านหลังกระดาษจากขวามาซ้าย

### ตัวอย่าง (ภาษาอังกฤษ)
```text
==================================================
  ตัวแปลงอักษรเบรลล์ภาษาอังกฤษ
  English Braille Translator
==================================================

กรุณาพิมพ์ข้อความภาษาอังกฤษ: noun

ผลลัพธ์อักษรเบรลล์:
----------------
 |** |*_ |*_ |**
 |_* |_* |__ |_*
 |*_ |*_ |** |*_
----------------

ผลลัพธ์อักษรเบรลล์: (mirror)
----------------
**| _*| _*| **|
*_| __| *_| *_|
_*| **| _*| _*|
----------------
```

### ตัวอย่าง (ภาษาไทย)

```text
==================================================
  ตัวแปลงอักษรเบรลล์ภาษาไทย
  Thai Braille Translator
  (อ้างอิง: liblouis th-g0.utb)
==================================================

กรุณาพิมพ์ข้อความภาษาไทย: กขค

ผลลัพธ์อักษรเบรลล์:
------------------------
**| _*| _*|
**| __| __|
__| _*| **|
------------------------

ผลลัพธ์อักษรเบรลล์: (mirror)
------------------------
 |** |*_ |*_
 |** |__ |__
 |__ |*_ |**
------------------------
```
